      Update the test's HTML equality comparison to not modify the content of
<pre> elements. This ensures that the test considers whitespace and
newline differences in the expected and the actual rendered HTML for
preformatted text.

erroneous newline after a <code> tag nested inside a <pre> element.

code blocks includes an extra blank line at the start.

written, there should be no newline after a <code> tag nested inside a
<pre> tag.
